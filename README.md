Приложение для быстрой передачи файлов и папок между компьютерами в одной локальной сети. Работает без интернета, не требует настройки и регистрации.

## Возможности

- Передача нескольких файлов и папок одновременно
- Drag & Drop - перетаскивание файлов прямо в окно приложения
- Автоматическое обнаружение пользователей в сети
- Отправка одному или нескольким получателям одновременно
- Сохранение структуры папок при передаче
- Подтверждение приёма файлов получателем
- Тёмная тема оформления
- Не требует интернета - работает только по локальной сети

## Системные требования

- Windows 10/11
- Python 3.8 или выше (для запуска из исходного кода)
- Локальная сеть (компьютеры должны быть в одной подсети)

## Установка

### Готовый исполняемый файл

Скачайте последнюю версию из раздела [Releases](https://github.com/ven1ck/p2p/releases/tag/release) на GitHub:

- [Скачать .exe файл](https://github.com/ven1ck/p2p/releases/tag/release)

Запустите скачанный файл - установка не требуется.

### Запуск из исходного кода

1. Установите Python 3.8 или выше с [python.org](https://www.python.org/downloads/)

2. Скачайте исходный код:
```bash
git clone https://github.com/ven1ck/p2p.git
cd ваш-репозиторий
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Запустите приложение:
```bash
python gui.py
```

## Как использовать

### Первый запуск

1. Запустите приложение на всех компьютерах, между которыми нужно передавать файлы
2. Дождитесь появления других пользователей в списке "Обнаруженные пользователи" (обычно 2-5 секунд)
3. Если пользователь не появился - проверьте, что компьютеры в одной сети и брандмауэр не блокирует приложение

### Отправка файлов

1. Добавьте файлы одним из способов:
   - Перетащите файлы или папки в окно приложения
   - Нажмите "Добавить файлы" для выбора файлов
   - Нажмите "Добавить папку" для выбора папки

2. Отметьте получателей в списке слева (можно выбрать нескольких)
   - Используйте "Выбрать всех" для отправки всем пользователям

3. Нажмите "Отправить выбранным"

4. На компьютере получателя появится запрос на подтверждение загрузки
   - После подтверждения начнётся передача
   - Файлы сохраняются в папку `received` в директории приложения

### Получение файлов

1. Запустите приложение
2. Когда отправитель начнёт передачу, появится окно с запросом
3. Просмотрите список файлов и их размер
4. Нажмите "Да" для подтверждения или "Нет" для отклонения
5. Принятые файлы сохранятся в папке `received`

## Сборка исполняемого файла

Если вы хотите собрать .exe файл самостоятельно:

1. Установите PyInstaller:
```bash
pip install pyinstaller
```

2. Подготовьте иконку (файл `logo.ico` в корне проекта)

3. Соберите приложение:
```bash
pyinstaller --onefile --windowed --clean --icon=logo.ico --name FileTransfer gui.py
```

4. Готовый файл появится в папке `dist/FileTransfer.exe`

## Настройки

### Изменение имени устройства

По умолчанию используется имя компьютера. Для изменения:
1. Введите новое имя в поле "Имя устройства"
2. Нажмите "Применить"
3. Другие пользователи увидят новое имя через несколько секунд

### Порты и директории

По умолчанию приложение использует:
- TCP порт 9999 - для передачи файлов
- UDP порт 50000 - для обнаружения пользователей
- Папка `./received` - для сохранения полученных файлов

Для изменения портов отредактируйте параметры в `core.py` (класс `FileTransferCore`).

## Решение проблем

### Пользователи не видят друг друга

1. Убедитесь, что компьютеры в одной локальной сети
2. Проверьте настройки брандмауэра - разрешите приложению входящие подключения на портах 9999 (TCP) и 50000 (UDP)
3. Попробуйте временно отключить антивирус
4. Перезапустите приложение на всех компьютерах

### Передача не начинается

1. Убедитесь, что получатель подтвердил загрузку в появившемся окне
2. Проверьте, что на диске получателя есть свободное место
3. Убедитесь, что файлы не открыты в других программах

### Файлы не сохраняются

1. Проверьте права доступа к папке приложения
2. Убедитесь, что антивирус не блокирует создание файлов
3. Проверьте путь к папке `received` - она должна существовать

### Drag & Drop не работает

1. Убедитесь, что установлен пакет `tkinterdnd2`:
```bash
pip install --force-reinstall tkinterdnd2
```
2. Перезапустите приложение
3. Попробуйте перетащить файлы в область списка файлов

## Безопасность

- Передача происходит только в локальной сети
- Файлы не загружаются на внешние серверы
- Получатель должен явно подтвердить загрузку каждого пакета файлов
- Приложение не собирает и не передаёт никакие данные
